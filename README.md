# UMC Pulse - 現代化桌面工具與排程小工具

**UMC Pulse** 是一款專為提升辦公效率設計的桌面小工具。結合了 Windows 11 的原生視覺美學與工業風的實用功能，讓您在忙碌的工作中能快速開啟常用程式、追蹤重要待辦事項，並獲得強制的置頂提醒。

---

## 🚀 主要功能

### 1. 常用功能區 (Functions)
- **快速啟動**：自定義多個常用網頁連結或內部程式路徑。
- **排程提醒**：可為功能設定「點擊提醒」時間，時間一到會跳出置頂視窗提醒您點擊，不再漏掉每日固定作業。
- **圖示化管理**：支援自定義 PNG 圖示，搭配自動換行排版，名稱再長也能完整呈現。

### 2. 待辦事項 (Tasks)
- **多樣排程**：支援「單次日期」、「每週固定週期」或「永遠顯示」三種任務模式。
- **強力提醒**：時間到達時會發送「系統托盤通知」並彈出「強制置頂對話框」，確保您能即時處理重要會議或事項。

### 3. 高級顯示控制
- **Windows 11 原生美感**：深色模式介面、半透明效果與微軟正黑體，視覺清晰且專業。
- **自由調整**：可隨意拖動視窗位置，系統會自動記錄您最後停放的位置。
- **釘選模式 (Pin)**：支援置頂模式切換，讓小工具始終顯示在所有視窗最前方。
- **系統托盤整合**：可隨時縮小至右下角通知區域，並透過右鍵選單快速操作。

## 📂 檔案清單說明

如果您是在公司環境使用，請根據權限選擇啟動方式：

1. **`UMCPulse_v3.exe`** (建議)：
   - 已打包好的單一執行檔，包含所有環境，直接執行即可。
   - 適合個人開發或允許執行 EXE 的環境。
2. **`UMC_Pulse.ps1`** (安控版)：
   - 專為禁止 EXE/Python 的嚴格公司環境設計。
   - 使用 PowerShell 原生腳本撰寫，雙擊 `Launch_UMC_Pulse.bat` 即可無痛啟動。
3. **`config.json`**：
   - 儲存所有功能設定與任務的設定檔，移動工具時請連同此檔一起移動。
4. **`Logo.png`**：
   - 工具的圖標檔案。

## 🛠️ 如何自定義？

### 新增功能/任務
- 點擊標題列旁的 **「＋」** 按鈕。
- 輸入名稱、目標（連結/路徑）與提醒時間。

### 編輯與刪除
- 在功能圖示或任務項目上 **「點擊滑鼠右鍵」**。
- 選擇「編輯內容」或「刪除」。

### 設定視窗
- 右擊右下角通知區域的 Logo 圖示，或在小工具上右擊選擇「設定」。
- 可開啟「開機自動啟動」或「啟動時縮小」。

## 💡 使用小撇步

- **移動視窗**：按住最上方的黑色標題列即可拖曳。
- **快速關閉/隱藏**：點擊右上角的 `—` 縮小至托盤，點擊 `X` 完全退出程式。
- **瘦身版 EXE**：`v3` 版本已經過精簡打包，僅包含核心組件，省電且省空間。
