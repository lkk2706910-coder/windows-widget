# UMC Pulse - 現代化桌面排程小工具 (v3 瘦身版)

**UMC Pulse** 是一款專為提升辦公效率設計的桌面導航工具。它將「常用列表」與「待辦提醒」整合在一個具備 Windows 11 美感的小視窗中，讓您不再漏掉重要會議與固定工作。

---

## 🚀 核心功能

### 1. 快捷功能區 (Functions)
- **快速啟動**：自定義一鍵開啟網頁連結、資料夾或內部程式。
- **點擊提醒**：可為特定功能設定「提醒時間」，時間一到會彈出置頂視窗，提醒您點擊該功能執行作業。
- **動態排版**：一排 4 個圖示的高效率排列，名稱過長會自動換行與顯示預設時間。

### 2. 待辦事項 (Tasks)
- **多樣排程**：支援單次提醒、每週固定日期提醒。
- **強力彈窗**：時間一到會發送氣泡通知，並跳出「強制置頂」對話框，直到您點擊確定為止。

### 3. 高品質設計
- **Windows 11 原生視覺**：深色模式背景與微軟正黑體，視覺清晰專業。
- **釘選功能 (Pin)**：可切換視窗是否始終顯示在所有視窗前方。
- **自動記憶位置**：隨意拖曳視窗後，工具會自動記住最後停留的位置。
- **系統托盤圖示**：點擊右上角 `—` 可縮小至右下角通知區，點擊 `X` 完全退出。

---

## 📂 使用說明

### 啟動工具
1. 直接執行 **`UMCPulse_v3.exe`** 即可。
2. 工具啟動後會自動讀取同目錄下的 `config.json`（您的所有設定與任務都儲存在此）。
3. 如果您需要將工具移至新電腦，請連同 `UMCPulse_v3.exe` 與 `config.json` 一起搬移。

### 如何自定義？
- **新增內容**：點擊小工具標題列旁的 **「＋」** 按鈕。
- **編輯與刪除**：在圖示或事項上點擊 **「滑鼠右鍵」**，即可進行編輯或刪除。
- **設定與開機啟動**：在視窗上點擊右鍵或在托盤圖示點擊右鍵，選擇「設定」即可開啟「開機自動啟動」功能。

### 常用技巧
- **移動位置**：按住最上方的黑色標題列即可進行拖曳。
- **強制置頂**：點擊標題列右方的 `[P]` 按鈕，可切換置頂(Pin)或非置頂模式。
